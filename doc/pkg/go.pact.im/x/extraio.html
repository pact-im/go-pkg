		rotate performs in-place left rotation of the slice by n positions from the
		start for positive n. If n is negative, the slice is right rotated by the
		absolute value of n.
		
		Examples:
		
			rotate([]byte("lohel"), 2)
			    => []byte("hello")
		
			rotate([]byte("45123"), 2)
			    => []byte("12345")
		
			rotate([]byte("34512"), -2)
			    => []byte("12345")

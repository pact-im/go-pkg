Package-Level Functions (total 12, in which 9 are exported)


	
		CancelOnError indicates that task group should return on non-nil error. This
		is the same behavior as the golang.org/x/sync/errgroup package.


	
		CancelOnReturn indicates that task group should be canceled if any
		subtask returns.


	
		Contextify adds context cancellation support to the task using the given
		shutdown function.


	
		Named returns a task that returns an error prefixed with name on failure.


	
		NeverCancel indicates that task group should not be canceled until
		all subtasks complete.


	
		Parallel returns a function that executes a group of tasks in parallel using
		the given cancellation condition. The resulting Task returns combined errors
		for all failed subtasks.


	
		ParallelExecutor returns an Executor instance for parallel execution.


	
		Sequential returns a function that executes a group of tasks sequentially
		using the given cancellation condition. The resulting Task returns combined
		errors for all failed subtasks.


	
		SequentialExecutor returns an Executor instance for sequential execution.
